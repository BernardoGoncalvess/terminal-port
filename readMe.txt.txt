Funcionalidades:
1 - Agir como um terminal convencional
2 - Comandos:
    - whoisme: Breve texto "sobre mim";
    - myprojects: Links para projetos meus (<a>);
    - email: Mostra o meu email para uso profissional;
    - help: Para mostrar os comandos inicialmente ao utilizador;
    - clear: Limpar tudo;
    - banner: Mostrar banner
3 - Animação de letras a aparecerem;

HTML:
1 - Criar um <div> para englobar tudo que preencha a página toda;
2 - Um <div> para o histórico;
3 - Meter um <input> que é onde o user mete os comandos;
4 - Prefixo antes do input;

OPCIONAL: Se não conseguir adicionar ASCII no JS, adiciono em HTML;
